⚙️  КАК ИСПОЛЬЗОВАТЬ ПРОГРАММУ
═════════════════════════════════════════════════════════════════════════════════

1️⃣  ЗАПУСК
    Дважды кликните на run.bat

2️⃣  ВЫБОР ВИДЕО
    В консоли появится список видео в папке
    Введите номер видео которое хотите обработать

3️⃣  ВЫБОР ПАРАМЕТРОВ
    Программа будет спрашивать параметры в следующем порядке:
    
    ▪ Диапазон кадров (обычно: 1 - все видео)
    ▪ Сила видео-шума (обычно: 3 - стандартный)
    ▪ Множитель силы (обычно: 3 - средний)
    ▪ Частота применения (обычно: 1 - каждый кадр)
    ▪ Аудио-маскировка (обычно: 1 - без маскировки)

4️⃣  ОБРАБОТКА
    Программа начнет обрабатывать видео
    Может занять от 5 минут до нескольких часов
    (зависит от длины видео и параметров)

5️⃣  РЕЗУЛЬТАТ
    После обработки появится файл: [имя]_protected.mp4
    Этот файл защищен от CV и ASR моделей


💡 РЕКОМЕНДУЕМЫЕ ПАРАМЕТРЫ
═════════════════════════════════════════════════════════════════════════════════

ДЛЯ ЯРКИХ ВИДЕО (видно даже при слабой защите):
  • Сила видео: 2 (Слабый - 0.070)
  • Множитель: 1 (Минимальный - 0.3x)
  → Результат: практически невидимо

ДЛЯ ОБЫЧНЫХ ВИДЕО:
  • Сила видео: 3 (Стандартный - 0.120) ← рекомендуется
  • Множитель: 3 (Средний - 1.0x)
  → Результат: слабо видны артефакты

ДЛЯ ЧЕРНЫХ/ТЕМНЫХ ВИДЕО:
  • Сила видео: 4 (Сильный - 0.180)
  • Множитель: 4 (Высокий - 1.4x)
  → Результат: хорошо видны артефакты

ЧАСТОТА:
  • Обычно: 1 (каждый кадр) ← РЕКОМЕНДУЕТСЯ
  • Если медленно: 2 (каждый 5-й кадр)
  • Если очень медленно: 3 (каждый 10-й кадр)


📂 ГДЕ НАЙТИ ГОТОВОЕ ВИДЕО
═════════════════════════════════════════════════════════════════════════════════

После обработки файл появится в ЭТОЙ ЖЕ ПАПКЕ где находится исходное видео.

Имя файла: [исходное_имя]_protected.mp4

Например:
  Было: video.mp4
  Стало: video_protected.mp4


⚡ БЫСТРАЯ ОБРАБОТКА (БЕЗ МЕДЛЕННОЙ ЧАСТИ)
═════════════════════════════════════════════════════════════════════════════════

Если программа обрабатывает очень долго:

1. Выберите частоту обработки: 3 (каждый 10-й кадр)
   → Обработка будет быстрее в 10 раз

2. Или выберите меньший диапазон кадров
   (например только первые 100 кадров)

⚠️  ПРОБЛЕМЫ И РЕШЕНИЯ
═════════════════════════════════════════════════════════════════════════════════

ПРОБЛЕМА: "run.bat ничего не открывает"
РЕШЕНИЕ:
  • Убедитесь что файл называется именно run.bat (малые буквы)
  • Пересоздайте файл run.bat если нужно
  • Проверьте что в папке есть файл media_cleaner.exe

ПРОБЛЕМА: "Черное окно консоли и все"
РЕШЕНИЕ:
  • Это нормально, ждите 20-30 секунд
  • Программа инициализируется
  • Появится меню выбора видео

ПРОБЛЕМА: "Windows Defender блокирует программу"
РЕШЕНИЕ:
  • Это нормально для скачанных программ
  • Нажмите "Еще информации" → "Все равно запустить"
  • Или добавьте исключение в Windows Defender:
    - Windows Defender → Управление параметрами
    - Добавить исключение → выберите папку программы

ПРОБЛЕМА: "Видео не найдено"
РЕШЕНИЕ:
  • Поместите видео файл В ПАПКУ программы
  • Или В ПОДПАПКУ программы
  • Поддерживаемые форматы: mp4, mov, avi, mkv, webm

ПРОБЛЕМА: "Программа работает очень долго"
РЕШЕНИЕ:
  • Это нормально для обработки видео
  • При большом разрешении может занять часы
  • Попробуйте:
    - Выбрать частоту 3 (каждый 10-й кадр)
    - Выбрать меньший диапазон кадров
    - Обработать только часть видео

ПРОБЛЕМА: "Ошибка при обработке"
РЕШЕНИЕ:
  • Проверьте что:
    - Видео файл не повреждён
    - Видео файл полностью скопирован
    - Достаточно свободного места на диске (>1 GB)
  • Попробуйте обработать другое видео
  • Если проблема повторяется - скопируйте сообщение об ошибке


🎯 ПРИМЕР ИСПОЛЬЗОВАНИЯ
═════════════════════════════════════════════════════════════════════════════════

1. Кликаю на run.bat
   → Появляется консоль

2. Вижу список видео:
   1 | video1.mp4
   2 | video2.mp4

3. Выбираю: 1
   → Выбран video1.mp4

4. Программа спрашивает параметры

5. Выбираю: 1, 2, 1, 1, 1
   (диапазон все видео, сила слабая, множитель минимальный, 
    частота каждый кадр, аудио без маскировки)

6. Жду обработку (20 минут для 10-минутного видео)

7. Видеть сообщение:
   ✓ ГОТОВО! Конечный файл: video1_protected.mp4

8. Открываю папку - там лежит новый файл video1_protected.mp4
