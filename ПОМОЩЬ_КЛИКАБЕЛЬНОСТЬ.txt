ЕСЛИ ВИДИТЕ КРАСНЫЕ ОШИБКИ В КОНСОЛИ:
──────────────────────────────────────

Это значит:
• CSS файл не загружен
• JavaScript файл не загружен  
• Сервер не работает

Решение:

1. Закройте ВСЕ вкладки браузера с 127.0.0.1 и localhost

2. В командной строке запустите заново:
   ► Откройте cmd.exe
   ► Перейдите в папку Media Cleaner
   ► Введите: python web_interface.py
   ► Вы должны увидеть: "Running on http://0.0.0.0:5000"

3. Откройте браузер → http://127.0.0.1:5000

4. Нажмите Ctrl+Shift+R (жесткая перезагрузка)
