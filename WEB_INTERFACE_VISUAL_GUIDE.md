## Интерактивные элементы

### Слайдер Epsilon
```
Epsilon (сила возмущения)
─────●────────────────── 0.12
[0.04 ──────────────────── 0.20]

При наведении: ──●───── (точка становится больше)
При перетаскивании: синхронизация с текстовым полем
```

### Выпадающий список Audio Level
```
Уровень защиты аудио:
[Слабый ▼]

Варианты:
├─ Отключено
├─ Слабый ✓
├─ Средний
└─ Сильный
```

### Кнопка "Начать обработка"
```
До выбора файла:      После выбора:
[🚀 НАЧАТЬ] (серая)   [🚀 НАЧАТЬ] (синяя) ← нажмаемая

При наведении: становится ярче и приподнимается
При клике: запускается загрузка
```

## Анимации

### Пульсирующая точка статуса
```
🟢 (яркая)  →  🟢 (бледная)  →  🟢 (яркая)
    ↓
Повторяется каждые 2 секунды
```

### Слайд-ин логов в консоли
```
Новый лог появляется:
[12:34:56] ✨ Сообщение  ← скользит слева направо
             ↓ (за 0.3 сек)
```

### Прогресс-бар загрузки
```
░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  0%
```
